Projects

Unsupervised Image Classification of Fish without the Inference of Cluster Number (Thesis)

In an extension of a previous collaboration with Innovasea, a deep learning framework was implemented for academic research. This work entailed the application of the DeepDPM clustering algorithm to the Fish4-Knowledge dataset for species classification and rare fish identification. The implementation, which used the ViT MAE and Real-ESRGAN for image enhancement, underwent several iterations to improve accuracy and clustering performance, ultimately achieving an 80% efficiency rate. The results, assessed via a visualization dashboard, pointed to the possibility of automated fish monitoring and species identification. This endeavor culminated in a master's thesis that provided a detailed understanding of performance metrics.

Understanding and expediting learning in Convolution Neural Networks using feature maps

In a cloud-based project involving MNIST and Fashion-MNIST datasets, distinctive feature maps were extracted from the preliminary layers of a convolutional neural network. This was achieved by utilizing cosine similarity, which allowed the identification of similar feature maps, enhancing the learning process by restricting the training loop to unique maps only. As a result, the model's optimization was significantly accelerated, surpassing standard approaches in accuracy and speed. The project implementation incorporated a CI/CD pipeline and was deployed on the AWS cloud, employing S3 buckets and SageMaker.

Song Popularity Prediction and Recommendation

Utilizing Spotify's song data from 1920-2021, a machine learning model was developed to predict song popularity and classify music genres, using k-means clustering and cosine similarity measurements. This approach involved thorough data analysis using song metadata, alongside a D3.js-based visualization tool to interpret model outputs. The project culminated in a recommendation system, using clustering results and cosine similarity to suggest similar music to users.

Stock price prediction using sentiment analysis of tweets

Leveraging yfinance and snscrape APIs, stock data and related news were scraped from Yahoo Finance and Twitter for various companies. This data was then analyzed using NLP techniques to preprocess the text, after which sentiment scores were extracted using a transformers-based FinBERT. Finally, deep learning models (LSTM and GRU) were constructed and their results compared to an ARIMAX-based time-series model.
